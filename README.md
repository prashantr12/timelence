# Timelence

A modern, responsive 5-page static website for Timelence — a technology company building software products, websites, mobile apps, web apps, and providing AI and consulting services.

## Pages
- Home (`index.html`)
- About (`about.html`)
- Products (`product.html`)
- Services (`services.html`)
- Contact (`contact.html`)

## Local preview
Open `index.html` in your browser, or serve the folder:

```bash
python3 -m http.server 5173
# then open http://localhost:5173
```

## Deploy
Upload these static files to any static hosting (GitHub Pages, Netlify, Vercel, S3, etc.) and point your domain to it.
